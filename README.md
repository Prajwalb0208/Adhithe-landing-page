## Local Development

Just open `index.html` in your browser or use a local server:

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

## Deployment

```bash
npm install
npm run deploy
```

Then configure GitHub Pages to use the `gh-pages` branch.
